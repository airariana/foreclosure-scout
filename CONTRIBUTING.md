# Contributing to Foreclosure Scout

Thanks for your interest in contributing to Foreclosure Scout!

## How to Contribute

### Reporting Issues

Found a bug? Have a suggestion? Open an issue!

**Bug Reports** should include:
- Description of the problem
- Steps to reproduce
- Expected behavior
- Actual behavior
- Browser/device information
- Screenshots if helpful

**Feature Requests** should include:
- Clear description of the feature
- Use case / why it's valuable
- Examples from other apps (if applicable)

### Code Contributions

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes**
4. **Test thoroughly** across browsers
5. **Commit**: `git commit -m "Add: description of your changes"`
6. **Push**: `git push origin feature/your-feature-name`
7. **Open a Pull Request**

### Development Guidelines

#### Code Style

- Use consistent indentation (2 spaces)
- Add comments for complex logic
- Keep functions small and focused
- Use meaningful variable names

#### Testing

Test your changes on:
- Chrome (desktop and mobile)
- Safari (desktop and iOS)
- Firefox
- Edge

#### PWA Testing

- Test offline functionality
- Verify service worker updates
- Check manifest validation
- Test installation flow

### Areas for Contribution

We especially welcome help with:

- 🎨 **Design/UI improvements**
- 📱 **Mobile optimizations**
- 🔍 **Additional data sources**
- 🤖 **AI analysis features**
- 📊 **Analytics and reporting**
- 🌐 **Accessibility enhancements**
- 📝 **Documentation**
- 🐛 **Bug fixes**

### Questions?

Open a discussion or issue if you need clarification!
